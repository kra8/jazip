# JaZip
郵便番号から住所を検索するCLIツール。

``` sh
$ jazip 2790031
千葉県,浦安市,舞浜
```

# Manual install
🚨macOS only  

```
$ curl -OL https://github.com/kra8/jazip/releases/download/v0.1.0/jazip-0.1.0.tar.gz
$ tar -xzvf jazip-0.1.0.tar.gz
$ sudo mv jazip-0.1.0/jazip /usr/local/bin/jazip
$ jazip <postcode>
```
